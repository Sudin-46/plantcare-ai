# 🌿 PlantCare AI

## 📌 Project Title
Detection of Nutrient Deficiency in Plants using Deep Learning

## 📖 Description
PlantCare is an AI-powered web application that detects plant nutrient deficiencies using leaf images.

It uses a deep learning model (MobileNetV2) to classify deficiencies such as:
- Nitrogen
- Phosphorus
- Potassium
- Magnesium
- Iron

The system also allows users to enter symptoms and combines both image + text analysis for better accuracy.

---

## 🚀 Features
- Upload leaf image
- Detect nutrient deficiency using AI
- Enter symptoms manually
- Combined analysis (image + text)
- Real-time result display
- Guidance for treatment

---

## 🛠️ Technologies Used
- Python (Flask)
- TensorFlow / Keras
- HTML, CSS, JavaScript
- Machine Learning / Deep Learning

---

## ▶️ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
